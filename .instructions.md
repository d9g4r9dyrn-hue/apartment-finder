# Apartment POC — Development Instructions

## Project Overview
**Apartment Finder POC** — A rental property aggregation system that:
- Scrapes apartment listings from multiple sources (Craigslist, Zillow, Apartments.com, Trulia, HotPads)
- Filters results by distance (5-mile radius from target location)
- Downloads property photos
- Generates an interactive HTML summary table with sorting/filtering

**Target Location**: Bluestone on the Bay, 19135 US Hwy 19 N, Clearwater, FL 33764  
**Search Radius**: 5 miles  
**Canonical Data**: `outputs/units.json`

## Architecture

### Directory Structure
```
apartment_poc/
├── config.json                    # Target location & search settings
├── outputs/
│   ├── units.json                # Aggregated listings database
│   ├── units-summary.html        # Generated browseable summary
│   ├── apartments/               # Per-unit detail pages
│   │   └── unit-XXXX/index.html
│   └── photos/                   # Downloaded property photos
│       └── unit-XXXX/photo-N.jpg
├── scripts/
│   ├── generate-html.py          # HTML generator (with distance filtering)
│   ├── crawl_all.py              # Master crawler (runs all scrapers)
│   └── scrapers/
│       ├── common.py             # Shared utilities
│       ├── craigslist.py         # ✓ Reliable scraper
│       ├── zillow.py             # ⚠️ Often blocked
│       ├── apartments_com.py     # ⚠️ JS-heavy
│       ├── trulia.py             # ⚠️ Similar to Zillow
│       └── README.md             # Scraper documentation
```

## Quick Start

### 1. Generate HTML Summary (with distance filtering)
```bash
python scripts/generate-html.py
# Output: outputs/units-summary.html
# Only shows units within 5 miles of target location
```

### 2. Run All Scrapers
```bash

# Runs: Craigslist, Zillow, Apartments.com, Trulia in sequence
# Updates: outputs/units.json and outputs/photos/
```

### 3. Run Single Scraper
```bash
python scripts/scrapers/craigslist.py  # Most reliable
# Other sources: zillow.py, apartments_com.py, trulia.py
```

## Key Concepts

### Distance Filtering
- **Implementation**: Haversine formula in `generate-html.py`
- **Config**: `config.json` defines target_location (lat/lon) and search_radius_miles
- **Filtering**: Automatic when generating HTML; lists outside radius are excluded
- **Display**: Distance column shows miles to target location
- **Filter Note**: Summary banner shows how many units were filtered out

### Photo Management
- **Storage**: `outputs/photos/unit-XXXX/photo-N.jpg`
- **References**: Stored as relative paths in `units.json` after "outputs/"
- **HTML Paths**: 
  - Summary page: `photos/unit-XXXX/photo-N.jpg`
  - Detail pages: `../../photos/unit-XXXX/photo-N.jpg`

## Development Guidelines

### Encoding
**CRITICAL**: Windows environment requires explicit UTF-8 encoding for Python file operations:
```python
UNITS_JSON.read_text(encoding='utf-8')
path.write_text(content, encoding='utf-8')
```
Without this, emojis and special characters cause UnicodeDecodeError.

### Scraper Implementation
All scrapers follow this pattern:
1. Load or create `outputs/units.json`
2. Fetch listings from source
3. Extract: address, price, beds, baths, sqft, lat/lon (if available), photos
4. Download photos to `outputs/photos/{unit_id}/`
5. Store relative paths in units.json
6. Check for duplicates by source_url
7. Add respectful delays (2+ seconds between requests)
8. Save to units.json

### Adding New Sources
1. Create `scripts/scrapers/{source_name}.py`
2. Implement scraper following `craigslist.py` pattern
3. Add to `SCRAPERS` dict in `crawl_all.py`
4. Document in `scripts/scrapers/README.md`

## Known Issues & Workarounds

### Scraper Blocking
- **Zillow**: Actively blocks scrapers. Needs Playwright + headless browser
- **Apartments.com**: Heavy JS; needs Playwright for JS rendering
- **Trulia**: Similar to Zillow; often returns no results

**Workaround**:
```bash
pip install playwright
pip install --upgrade beautifulsoup4
# Then run: python scripts/crawl_all.py
```

### Python Requirements
- `requests` — HTTP requests
- `beautifulsoup4` — HTML parsing
- `playwright` (optional) — JS rendering for blocked sites

Install:
```bash
pip install requests beautifulsoup4
pip install playwright  # Optional, for JS sites
```

### No Photos on Windows
If thumbnails don't display:
1. Check that `outputs/photos/` was copied
2. Verify file paths are relative (no absolute paths in HTML)
3. Ensure UTF-8 encoding was used when writing HTML

## Important Context for Future Work

### Why Distance Filtering Exists
Original issue: Scraper included Apollo Beach (47 miles away) even though requirement was 5 miles. The Haversine distance filter prevents this.

### Why Photo Paths Are Relative
Absolute paths break when files are moved or copied. Relative paths are portable across systems and file transfers.

### Why UTF-8 Encoding Is Required
Windows uses cp1252 by default, which can't encode emojis and special Unicode characters in the listing data.

### Why Multiple Scrapers
Different sources have different strengths. Craigslist is fast and reliable. Zillow/Apartments.com have more inventory but are harder to scrape. Aggregating all sources gives best coverage.
